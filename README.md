# TicTacToeAndroid
### Игра в крестики-нолики 3х3.  
Приложение для андроид для игры в крестики нолики.  
Можно играть как с ботом, так и с другим человеком(с одного устройства).  
Есть кнопка отката предыдущего шага.  
Есть кнопка рестарта.
